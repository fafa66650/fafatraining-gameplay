# FAFATRAINING GAME ARENA — V9 PRO

Application d'escape game dématérialisée avec deux modes séparés :

- Mode joueur : choix mission, public, difficulté, briefing, énigmes, progression.
- Mode administrateur : création de session, équipes, code session, guide réponses.

Compatible GitHub Pages : déposer tous les fichiers à la racine du dépôt.

Notes :
- Les sessions sont stockées en localStorage sur l'appareil qui les crée.
- L'audio utilise la synthèse vocale du navigateur.
- La mini-carte est prête côté interface. Pour un terrain réel, ajouter les coordonnées GPS exactes de chaque étape.
